# A-Beigner-Translator

A responsive web-based language translator built using HTML, CSS, and JavaScript. It supports multiple languages, text-to-speech with real-time word highlighting, OCR image text extraction, fast translation mode, and a clean modern glass UI. Designed for both desktop and mobile devices with smooth auto-scroll and speech control.
